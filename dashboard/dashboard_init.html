Анкета абитуриента

Для заполнения анкеты вам понадобятся следующие документы:

  • Паспорт (лицевая страница и страница c пропиской; стр. 19 со сведениями о предыдущем паспорте, если по нему был сдан ЕГЭ).
  • Документ об образовании (вкладыш, приложение с оценками полностью)
  • СНИЛС (при наличии)
  • Документы, подтверждающие льготу
  • Договор о целевом обучении (при поступлении на целевое).
